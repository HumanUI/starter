# Usage

### First step

``` bash
# install vue-cli
npm install -g vue-cli
```

``` bash
# Create you project
vue init freshesx/starter my-project
```

### Second step

``` bash
# Open my-project
$ cd my-project

# Use git init project, and yarn install
$ yarn run first

# Start development
$ yarn start
```
